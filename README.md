# Drug Design Optimization Using Genetic Algorithm

## Project Overview
This project focuses on optimizing drug-like molecules using Genetic Algorithms and cheminformatics techniques. Users provide molecular structures in SMILES format, and the system generates improved candidate molecules based on selected optimization criteria.

## Objectives
- Generate optimized molecular structures
- Improve desired chemical properties
- Apply crossover and mutation operations
- Evaluate molecule fitness scores
- Visualize optimization results

## Features
- SMILES input support
- Genetic Algorithm based optimization
- Mutation and crossover operations
- Fitness evaluation
- Result history in CSV format
- Graphical visualization of outputs

## Technologies Used
- Python
- Streamlit
- RDKit
- Pandas
- Matplotlib
- NumPy

## Modules
1. Input Module – Accepts SMILES strings  
2. Optimization Module – Applies GA operations  
3. Evaluation Module – Calculates fitness score  
4. Visualization Module – Displays graphs/results  
5. Export Module – Stores outputs in CSV files
